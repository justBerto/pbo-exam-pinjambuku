# pbo-exam-pinjambuku
Aplikasi java untuk melakukan peminjaman buku yang ada didalam database.

Untuk mengkoneksikan DB nya anda bisa mengubah kode di bagian `init.java`

Dibuat menggunakan Netbeans 8.2 (old version).
